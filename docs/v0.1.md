# v0.1 — GitHub Account Presentation Optimizer

v0.1 delivers a **CLI-only, read-only** analyzer that scans a GitHub account and outputs structured, explainable presentation scores per repository.

## What’s in v0.1

- **Scan**: List repos for a user (optional filter by repo name), fetch metadata and README, normalize, score.
- **Scoring dimensions**: Name clarity, description quality, topic coverage, README structure, activity recency, metadata hygiene.
- **Presets**: `indie-hacker`, `open-source-maintainer`, `portfolio-dev`, `enterprise-internal` (weights and expectations).
- **Output**: Table (default) or JSON.
- **Suggestions** (v0.2): `--mode suggest` adds advisory suggestions per repo; all advisory-only, no writes.

## Quick usage

```bash
# From repo root (after pip install -e ".[dev]")
export GITHUB_TOKEN=ghp_xxx
gh-visibility scan --user your-username --output table
gh-visibility scan --user your-username --preset open-source-maintainer --output json
gh-visibility scan --user your-username --mode suggest --output table
gh-visibility scan --user your-username --repo single-repo-name --benchmark internal
```

## Sample output (JSON)

Each evaluation includes `repo`, `analysis`, `scores` (per dimension + overall), and optionally `suggestions`. Use `--output json` and pipe to a file, then open `web/index.html` and paste the JSON to view the read-only dashboard.
